# 3D Models for Electric Vehicle Project

Place your GLB model files in this directory. The following models are needed:

## Required Models

1. **battery.glb** - Lithium-ion battery pack model
2. **motor.glb** - Electric motor model  
3. **braking.glb** - Regenerative braking system model

## Recommended Sources

### Free Models (CC0/CC-BY Licensed)

1. **Sketchfab** - Search for "battery", "electric motor", "brake" models
   - https://sketchfab.com/search?q=battery&type=models&features=downloadable&sort_by=-likeCount
   - https://sketchfab.com/search?q=electric+motor&type=models&features=downloadable
   - https://sketchfab.com/search?q=brake+disc&type=models&features=downloadable

2. **CGTrader Free Section**
   - https://www.cgtrader.com/free-3d-models/industrial/industrial-machine/battery

3. **Free3D**
   - https://free3d.com/3d-model/battery

## Download Instructions

1. Visit the recommended sources above
2. Filter by "Free" and "Downloadable" 
3. Download GLB format (or convert from other formats)
4. Rename files to match the required names above
5. Place files in this `models/` directory

## File Format

- **Format**: GLB (preferred) or GLTF
- **Size**: Keep under 5MB per file for optimal loading
- **License**: Ensure models are CC0 or CC-BY licensed for free use
